About XML

XML(eXtensible Markup Language) is a structured format to
using tags to organize and store data for easy
exchange between different systems

How XML is used

it is used to structure and store data in readable format
with tags,facilitating easy shairing and exchange of
information between different system and platforms

Example of XML

            <title>Harry Potter</title>

javascript

javascript is a dynamic programming language utilized for web development,
enabling interactive web elements,user interface enhancements,and
server-side scripting for versatile applications.

Javascript related to HTML and CSS

JavaScript complements HTML/CSS by enabling interactive
webpage elements, handling events, and manipulating content,
enhancing user experiences within
web applications and sites.

JavaScript properties,elements,methods

Properties refers to characteristics or attributes
aissociated with JavaScript objects, like the name,age,
or email properties within person object

Methods are functions associated with objects or data types,such
as the push() method used to add an element to an array.

Elements refer to HTML elements accessed and manipulated using
JavaScript,as shown by the getElementById() method accessing an HTML element with the id
main-heading and changing its style.
